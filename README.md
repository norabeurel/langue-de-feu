# langue de feu
It is a website for translating words into langue de feu, developed in PHP.
***

## Version
* PHP : 8.3.31
***

## Install


### Install Docker and Docker Compose 
[Install guide - Docker](https://docs.docker.com/engine/install/)

[Install guide - Docker Compose](https://docs.docker.com/compose/install/)

### Use docker 
Run the command
```
docker compose up
```
***

### Use composer
Run the command
```
composer install
```
***

## Access the interface
Go to a browser and enter http://localhost:8001/
